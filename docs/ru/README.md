# Lumin: Твой Язык Программирования Будущего
[![GitHub stars](https://img.shields.io/github/stars/Tosa5656/Lumin?style=social)](https://github.com/Tosa5656/Lumin)
[![GitHub forks](https://img.shields.io/github/forks/Tosa5656/Lumin?style=social)](https://github.com/Tosa5656/Lumin)
[![GitHub issues](https://img.shields.io/github/issues/Tosa5656/Lumin)](https://github.com/Tosa5656/Lumin/issues)
[![License](https://img.shields.io/github/license/Tosa5656/Lumin)](https://github.com/Tosa5656/Lumin/blob/master/LICENSE.md)

**Lumin** - это новый, разрабатываемый язык программирования, созданный с акцентом на простоту, читаемость и производительность.  Он спроектирован так, чтобы быть универсальным инструментом для широкого круга задач, от небольших скриптов до крупных, сложных приложений. Lumin стремится предложить разработчикам интуитивно понятный синтаксис и мощные возможности, обеспечивая при этом безопасность и предотвращая распространенные ошибки.

## 💡 Основные принципы и философия
*   **Назначение:** Универсальный язык программирования.
*   **Простота:** Легкий в изучении и использовании.
*   **Читаемость:** Код должен быть легко читаемым и понятным.
*   **Производительность:** Достаточно производительный для широкого круга задач.
*   **Безопасность:** Предотвращение ошибок и обеспечение безопасности кода.

## 📚 Возможности языка
### Типы данных
#### Основные типы

*   `int`: Целые числа.
*   `float`: Числа с плавающей точкой.
*   `bool`: Булевы значения (`true`/`false`).
*   `string`: Строки.
*   `null`: Нулевое значение.

#### Структуры данных
*   **Массивы:**
    *   Могут быть фиксированного или динамического размера.
    *   Однородные (все элементы одного типа).
    *   Строгая типизация.

    ##### Создание:

    ```cpp
    int[length] array = new int[length]; // Cоздание массива фиксированной длинны
    int array = {1, 2, 3};
    int array = [1, 2, 3];
    ```

    ##### Операции:
    ```cpp
    array[index];        // доступ по индексу
    array.length();      // длина массива
    array.reverse();     // переворот
    array.sort();        // сортировка
    array.add(element);  // добавление элемента в конец (для динамических массивов)
    array.remove(index); // удаление элемента по индексу (для динамических массивов)
    array.find(element); // поиск элемента
    ```

*   **Списки:**
    ##### Реализация:
    *   Связный список.
    *   Однородные (все элементы одного типа).
    *   Строгая типизация.

    ##### Операции:
    ```cpp
    list.add(element);          // добавление в конец
    list.remove(index);         // удаление по индексу
    list.insert(index, element); // вставка по индексу
    list.get(index);            // получение элемента по индексу
    list.length();              // длина списка
    list.find(element);         // поиск элемента
    ```

  *   **Словари:**
    ##### Реализация:
    *   Хеш-таблица (Hash Table).
    *   Гибкая типизация: Тип ключа и тип значения определяются при добавлении первого элемента.
    ##### Операции:
    ```cpp
    dictionary[key] = value; // добавление/изменение
    value = dictionary[key]; // получение значения
    dictionary.remove(key);  // удаление пары
    dictionary.containsKey(key); // проверка наличия ключа
    dictionary.size();       // количество пар
    dictionary.keys();       // список ключей
    dictionary.values();     // список значений
    ```

*   **Строки:**
    *   Изменяемые (mutable).
    *   Поддержка регулярных выражений (детали реализации еще не определены).

    ##### Операции:
    ```cpp
    string1 + string2;                     // конкатенация
    string.length();                       // длина строки
    string[index];                         // доступ к символу по индексу
    string.find(substring);                // поиск подстроки
    string.replace(oldSubstring, newSubstring); // замена подстроки
    string.split(delimiter);               // разбиение строки
    string.toUpper(); || string.toLower();    // регистр
    string.trim();                         // удаление пробелов
    string.insert(index, substring);      // вставка подстроки
    string.remove(startIndex, length);     // удаление подстроки
    ```

### Управление памятью
*   Автоматическое управление памятью на основе владения (для локальных переменных).
*   Явное удаление объектов: `delete(object)`. Для пользовательских типов можно перегрузить `delete` для кастомной логики.
*   `setLifeTime(seconds)`: Устанавливает время жизни объекта в секундах.
*   Детектор циклических ссылок.

### Операторы
*   Арифметические: `+`, `-`, `*`, `/`, `%`.
*   Сравнения: `==`, `!=`, `>`, `<`, `>=`, `<=`.
*   Логические: `&&`, `||`, `!`.
*   Присваивания: `=`, `+=`, `-=`, `*=`, `/=`, `%=`.

### Управляющие конструкции
*   `if else`
    ```cpp
    int a = 5;
    int b = 3;

    if (a == b) // Output: a != b
    {
        print("a = b");
    }
    else
    {
          print("a != b");
    }
    ```

*   `for`
    ```cpp
    int array = {1, 2, 3, 4, 5, 6};
    for (int element : array) // Output: 1 2 3 4 5 6
    {
        print(element);
    }
    ```

*   `while`
    ```cpp
    int array = {1, 2, 3, 4, 5, 6};
    int i = 0;

    while (i < array.length()) // Output: 1 2 3 4 5 6
    {
        print(array[i]);
        i++;
    }
    ```

### Функции
*   Определение функций:
```cpp
    type function_name(arg_type arg, ... )
    {
        return optional type;
    }
```

*   Для возврата значений используется `return`.

### Обработка исключений
*   `try-catch-finally`
*   Возможность создания собственных типов исключений (наследование от базового класса `Exception`).
*   Необработанные исключения логируются и завершают программу.

### Модули и пространства имен
*   Каждый файл является модулем.
*   Импорт модулей: `%include "name"`.
*   Неймспейсы:

    *   Использование:

    ```cpp
    namespace MyNamespace
    {
        function Test();
        // Somethink...
    }

    MyNamespace::Test();
    ```

### Многопоточность
*   Поддержка потоков (threads).
*   Механизмы синхронизации: мьютексы (mutex), семафоры (semaphore), каналы (channel).
